# Manual da Marca — Alzirene Andrade da Fonseca

Landing page do manual de marca da Psicóloga Clínica / Neuropsicóloga / Psicanalista **Alzirene Andrade da Fonseca**.

Design: leveza silenciosa. Paleta lavanda, tipografia Grand Cru (serifada de alto contraste) + Mulish (apoio).

## Como abrir

Abra `Manual da Marca (Landing).html` em qualquer navegador moderno. As fontes (`assets/fonts/`) e as imagens (`assets/*.png`) são carregadas por caminhos relativos — não precisa de servidor.

Para hospedar em GitHub Pages, basta ativar Pages apontando para a branch `main` / raiz.

## Estrutura

```
.
├── Manual da Marca (Landing).html   # página completa
├── landing.css                      # estilos (fontes, tokens, todas as seções)
└── assets/
    ├── fonts/                       # Grand Cru (L + S) e Mulish
    ├── perfil.png                   # foto da profissional
    ├── simbolo.png                  # monograma AAF + cérebro + Ψ
    ├── logo-completa.png            # versão completa do logotipo
    └── logo-horizontal.png          # versão horizontal do logotipo
```
